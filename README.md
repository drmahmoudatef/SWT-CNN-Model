<h3>Model Overview</h3>
<p>This project implements a <b>CNN model enhanced with 4-channel Wavelet Transform (LL, LH, HL, HH)</b> for detecting tampered regions in images.  
The model is applied on 7 datasets covering both <b>Copy-Move</b> and <b>Splicing</b> tampering types.</p>
